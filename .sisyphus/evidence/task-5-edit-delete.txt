Edit and Delete button logic:

Edit comment (line 798-820):
- editComment(commentId, currentContent) function
- Uses prompt() for editing
- Validates length <= 1000
- PUT /api/comments/{commentId}
- Shows '无权操作' on FORBIDDEN, '编辑失败' on other errors
- Only shown to comment owner (isOwner check)

Delete comment (line 822-841):
- deleteComment(commentId) function
- Uses confirm() for confirmation
- DELETE /api/comments/{commentId}
- Shows '无权操作' on FORBIDDEN, '删除失败' on other errors
- Shown to comment owner OR admin (canDelete = isOwner || currentUser.is_admin)

Both functions reset pagination state and reload from page 1 after success.
